# 세무회계 이윤 AI 세무 챗봇

대구 달서구 세무회계 이윤의 AI 세무 상담 챗봇입니다.
OpenAI GPT-4o mini 기반으로 종합소득세, 상속세, 증여세 등 세무 질문에 답변합니다.

## 기능
- 종합소득세, 양도소득세, 상속세, 증여세, 부가세, 법인세 상담
- 빠른 질문 버튼
- 모바일 반응형 디자인
- 대화 히스토리 유지

## 사용법
1. 우측 상단 "API 설정" 버튼 클릭
2. OpenAI API 키 입력
3. 세무 관련 질문 입력 또는 빠른 질문 클릭

## 배포
Cloudflare Pages로 배포되어 있습니다.
