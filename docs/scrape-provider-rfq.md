# 신고서 스크래핑 제공사 견적 요청서 (RFQ)

> 작성: 2026-06-17 · 발주처: 대구 달서구 **세무회계 이윤** (대표세무사 이재윤)
> 용도: 자체 세무 챗봇·관리 플랫폼에 거래처 신고 데이터 연동
> 대상 제공사: **하이픈(HYPHEN)** 우선 · **CODEF(쿠콘)** 비교 · (틸코 백업)

## 0. 한 줄 요약
세무대리 수임 거래처 **약 200곳**의 **부가세·종소세·법인세 신고결과**를, 거래처 개인 인증정보를
저장하지 않고 **세무대리인 권한(수임동의) 기반**으로 자동 조회하려 함. 1단계 범위·물량 기준 견적 요청.

## 1. 범위 (1단계)
| 구분 | 포함 | 비고 |
|---|---|---|
| 홈택스 부가세 신고결과 | ✅ | 신고결과·수입금액·납부세액 |
| 홈택스 종소세(개인) 신고결과 | ✅ | 수입금액·결정세액 |
| 홈택스 법인세 신고결과 | ✅ | 법인 거래처 |
| 원천세 | ❌ (향후 확장) | 가능 여부만 문의 |
| 4대보험 | ❌ (현재 제외) | — |
| 위택스(지방세) | ❌ (현재 제외) | — |

## 2. 인증/권한 모델 (핵심)
- **세무대리인 권한 기반.** 사장님이 정식 세무대리인(수임)이므로, 거래처 개인 간편인증/인증서를
  매번 받지 않고 세무대리 권한으로 수임 거래처를 일괄 조회하고자 함.
- **거래처 인증정보(인증서·ID/PW)는 우리 측 저장 안 함.** 제공사가 인증 보유.

## 3. 물량 추정 (200거래처, 1단계)
| 항목 | 거래처당/년 | 합계/년(추정) |
|---|---|---|
| 부가세 신고결과 | 2~4회 | 400~800건 |
| 종소세/법인세 | 1회 | ~200건 |
| **합계** | | **약 600~1,500건/년** |

> 무인 월간 모니터링은 1단계 제외(신고철 전후 결과 조회 위주). 향후 확장 시 호출량 증가.

## 4. 견적·확인 질문 (회신 요망)
1. **세무대리인 권한으로 수임 거래처(200곳) 일괄 조회** 지원 여부 및 방식
2. 수임동의 **수집·해지 연동** 흐름 (API/대시보드)
3. 조회 항목 확정: 부가세·종소세·법인세 **신고결과/수입금액/납부세액** 제공 범위
4. **과금**: 건당 종량 단가 / 월정액 / 구간 할인 — 위 물량(연 600~1,500건) 기준 예상 비용
5. 초기 도입비·최소 약정 유무
6. 연동 방식: REST/JSON·웹훅, **샌드박스/데모** 제공 여부, 개발 문서
7. **신고철 대량조회 제한**(rate limit)·홈택스 과부하 대응
8. 개인정보 위탁·전자금융거래법 관련 **컴플라이언스 가이드/계약서** 제공 여부
9. (향후) **원천세** 확장 가능 여부 및 추가 단가
