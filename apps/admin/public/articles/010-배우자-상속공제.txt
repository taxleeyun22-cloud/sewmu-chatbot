<h2>배우자 상속공제의 요건과 계산</h2>

<p>상속이 발생하면 상속재산가액에서 채무 등을 차감하고 상속공제를 뺀 금액에 세율을 곱하여 상속세를 산정합니다. 피상속인의 배우자가 생존해 있다면 최소 5억원에서 최대 30억원까지 배우자 상속공제를 받을 수 있습니다.</p>

<hr>

<h3>법령 근거</h3>

<p>상속세및증여세법 제19조(배우자 상속공제)에서 배우자 상속공제의 요건과 한도를 규정하고 있습니다.</p>

<blockquote>상속세및증여세법 제19조 제1항: 거주자의 사망으로 상속이 개시되는 경우로서 배우자가 실제 상속받은 금액은 상속세 과세가액에서 공제한다. 다만, 그 공제액은 배우자의 법정상속분에 상당하는 금액을 한도로 하되, 최소 5억원을 공제한다.</blockquote>

<hr>

<h3>배우자 상속공제액 산정 공식</h3>

<div class="example-box">
<p>배우자 상속공제액 = MAX [ MIN(1, 2, 3), 5억원 ]</p>
<ul>
<li>1: 배우자가 실제 상속받은 금액</li>
<li>2: 배우자의 법정상속분 가액 (상속재산가액 x 법정상속비율 - 사전증여재산 등)</li>
<li>3: 30억원 (상한)</li>
</ul>
<p>즉, 배우자가 실제 상속받은 금액이 없더라도 최소 5억원은 공제됩니다.</p>
</div>

<hr>

<h3>배우자의 법정상속분 가액 계산</h3>

<p>민법 제1009조에 따라, 배우자는 직계비속과 공동 상속 시 직계비속의 상속분에 5할을 가산(1.5배)합니다.</p>

<table>
<tr><th>상속 구성</th><th>배우자 비율</th><th>기타 상속인 비율</th></tr>
<tr><td>배우자 + 자녀 1명</td><td>1.5 / 2.5 = 60%</td><td>1 / 2.5 = 40%</td></tr>
<tr><td>배우자 + 자녀 2명</td><td>1.5 / 3.5 = 42.86%</td><td>각 1 / 3.5 = 28.57%</td></tr>
<tr><td>배우자 + 자녀 3명</td><td>1.5 / 4.5 = 33.33%</td><td>각 1 / 4.5 = 22.22%</td></tr>
<tr><td>배우자만 (직계비속 없음)</td><td>단독 상속 100%</td><td>-</td></tr>
</table>

<hr>

<h3>구체적 계산 예시</h3>

<div class="example-box">
<p>사례 1: 배우자 + 자녀 2명, 상속재산 20억원</p>
<ul>
<li>배우자 법정상속분: 20억 x 1.5/3.5 = 약 8.57억원</li>
<li>배우자가 실제 10억원 상속받은 경우</li>
<li>공제액: MIN(10억, 8.57억, 30억) = 8.57억원</li>
<li>법정상속분(8.57억)이 한도이므로 8.57억원 공제</li>
</ul>
</div>

<div class="example-box">
<p>사례 2: 배우자 + 자녀 2명, 상속재산 20억원, 배우자 실제 상속 0원</p>
<ul>
<li>배우자가 실제 상속받은 금액이 0원</li>
<li>최소 공제: 5억원 공제 (실제 상속 없어도 최소 5억원 보장)</li>
</ul>
</div>

<div class="example-box">
<p>사례 3: 배우자 + 자녀 1명, 상속재산 50억원</p>
<ul>
<li>배우자 법정상속분: 50억 x 1.5/2.5 = 30억원</li>
<li>배우자가 실제 35억원 상속받은 경우</li>
<li>공제액: MIN(35억, 30억, 30억) = 30억원 (상한 적용)</li>
</ul>
</div>

<hr>

<h3>배우자 상속공제 신고기한</h3>

<p>배우자 상속공제는 상속세 신고기한(상속개시일이 속하는 달의 말일부터 6개월) 이내에 배우자의 상속재산을 분할(협의분할 또는 법원의 판결)하여 신고해야 합니다. 신고기한 내 분할이 완료되지 않으면 법정상속분 범위 내에서만 공제됩니다.</p>

<div class="example-box">
<p>배우자 상속재산 분할기한 특례: 상속세 신고기한 후 9개월 이내(즉, 상속개시일로부터 약 15개월)에 분할을 완료하면 배우자 상속공제를 적용받을 수 있습니다(상속세및증여세법 제19조 제2항).</p>
</div>

<hr>

<h3>관련 예규 및 판례</h3>

<ul>
<li>상증통 19-0...1: 배우자란 민법상 혼인관계에 의한 배우자를 의미하며, 사실혼 관계의 배우자는 공제 적용 불가</li>
<li>서면-2020-상속증여-4589 (2021.03.15.): 배우자가 상속을 포기한 경우에도 다른 상속인이 없으면 최소 5억원 공제 가능</li>
<li>대법원 2013두16896 (2015.06.11.): 배우자 상속공제 한도인 법정상속분 가액 산정 시 사전증여재산가액을 차감</li>
<li>조심-2019-서-2034 (2020.01.16.): 분할기한 내 미분할 시 법정상속분 범위 내에서만 공제 적용</li>
</ul>

<hr>

<h3>자주 묻는 질문</h3>

<p>Q. 배우자가 상속을 포기하면 배우자 상속공제를 받을 수 없나요?</p>
<p>A. 배우자가 상속을 포기하더라도 최소 공제 5억원은 적용됩니다. 다만, 실제 상속받은 금액이 없으므로 5억원을 초과하는 공제는 받을 수 없습니다.</p>

<p>Q. 사실혼 배우자도 배우자 상속공제를 받을 수 있나요?</p>
<p>A. 아닙니다. 배우자 상속공제에서 "배우자"란 민법상 혼인관계에 의한 법률상 배우자를 의미합니다. 사실혼 관계의 배우자에게는 배우자 상속공제가 적용되지 않습니다.</p>

<p>Q. 배우자 상속공제를 최대한 받으려면 어떻게 해야 하나요?</p>
<p>A. 배우자가 법정상속분만큼 실제로 상속받아야 합니다. 상속재산 분할협의 시 배우자에게 법정상속분 이상을 배분하고, 신고기한(또는 분할기한 특례 15개월) 내에 분할을 완료하여 신고해야 합니다.</p>

<hr>

<h3>주의사항 및 실수하기 쉬운 점</h3>

<ul>
<li>배우자 상속공제 최대한도 30억원을 받으려면 배우자가 실제로 30억원 이상 상속받아야 하며, 법정상속분도 30억 이상이어야 함</li>
<li>상속재산 분할기한(신고기한 후 9개월) 경과 시 배우자 실제 상속분 기준 공제가 제한됨</li>
<li>사전증여재산이 있는 경우 법정상속분 가액에서 차감되므로, 사전증여가 많으면 배우자 공제 한도가 줄어듦</li>
<li>배우자 상속공제를 크게 받기 위해 배우자에게 재산을 집중시키면, 배우자 사망 시(2차 상속) 상속세 부담이 증가할 수 있음</li>
</ul>

<hr>

<h3>절세 전략</h3>

<ul>
<li>1차 상속과 2차 상속(배우자 사망)을 종합적으로 고려하여 배우자 상속분 결정</li>
<li>배우자 상속공제와 일괄공제(5억원), 기타 인적공제를 조합하여 최적의 공제구조 설계</li>
<li>배우자에게 법정상속분만큼 상속 후 10년 경과 뒤 자녀에게 증여하면 2차 상속 시 합산 회피 가능</li>
<li>상속재산 분할협의 시 배우자 공제와 동거주택 상속공제(최대 6억원) 동시 적용 가능 여부 검토</li>
<li>상속세 신고기한 전에 반드시 분할협의를 완료하여 배우자 공제 극대화</li>
</ul>
