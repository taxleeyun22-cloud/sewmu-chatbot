<h2>주택임대사업자 재산세 감면</h2>

<p>주택임대사업자로 등록하면 전용면적에 따라 재산세를 50%에서 100%까지 감면받을 수 있습니다. 감면을 받으려면 지방자치단체에 임대사업자 등록을 하고, 면적/공시가격/임대료 증액 제한 등의 요건을 충족해야 합니다.</p>

<hr>

<h3>법령 근거</h3>

<p>지방세특례제한법 제31조의3 제2항 및 제31조의4에서 주택임대사업자에 대한 재산세 감면 요건과 감면율을 규정하고 있습니다.</p>

<blockquote>지방세특례제한법 제31조의4 제1항: 임대사업자가 대통령령으로 정하는 임대주택(의무임대기간 10년 이상, 임대료 증액 제한 등 요건 충족)으로 등록한 주택에 대해서는 재산세를 감면한다.</blockquote>

<hr>

<h3>구체적 계산 예시</h3>

<div class="example-box">
<p>사례 1: 전용 39m2 오피스텔 (공시가격 1.5억원)</p>
<ul>
<li>재산세 과세표준: 1.5억 x 60%(공정시장가액비율) = 9,000만원</li>
<li>재산세율(1억 이하): 0.1%</li>
<li>재산세: 9,000만원 x 0.1% = 90,000원</li>
<li>감면율: 100% (40m2 이하)</li>
<li>감면 후 재산세: 0원</li>
</ul>
</div>

<div class="example-box">
<p>사례 2: 전용 55m2 공동주택 (공시가격 3억원)</p>
<ul>
<li>재산세 과세표준: 3억 x 60% = 1.8억원</li>
<li>재산세: 1.5억 x 0.1% + 3,000만원 x 0.15% = 150,000 + 45,000 = 195,000원</li>
<li>감면율: 75% (40~60m2)</li>
<li>감면 후 재산세: 195,000 x 25% = 48,750원</li>
<li>연간 절감액: 146,250원</li>
</ul>
</div>

<div class="example-box">
<p>사례 3: 전용 75m2 공동주택 (공시가격 5억원)</p>
<ul>
<li>재산세 과세표준: 5억 x 60% = 3억원</li>
<li>재산세: 1.5억 x 0.1% + 1.5억 x 0.15% = 150,000 + 225,000 = 375,000원</li>
<li>감면율: 50% (60~85m2)</li>
<li>감면 후 재산세: 375,000 x 50% = 187,500원</li>
<li>연간 절감액: 187,500원</li>
</ul>
</div>

<h3>자주 묻는 질문</h3>

<p>Q. 아파트도 재산세 감면을 받을 수 있나요?</p>
<p>A. 취득세 감면과 달리, 재산세 감면은 아파트도 대상에 포함됩니다. 다만, 전용면적 85m2 이하, 공시가격 요건(수도권 6억 이하) 등을 충족해야 합니다.</p>

<p>Q. 재산세 감면은 자동으로 적용되나요?</p>
<p>A. 아닙니다. 지자체에 임대사업자로 등록하고, 감면 신청을 해야 합니다. 등록만 하고 감면 신청을 하지 않으면 일반 세율이 적용될 수 있으므로, 등록 후 관할 구청 세무과에 감면 신청을 반드시 확인하세요.</p>

<p>Q. 공시가격이 매년 오르면 감면이 중단될 수 있나요?</p>
<p>A. 맞습니다. 매년 6월 1일(재산세 과세기준일) 현재의 공시가격이 기준(수도권 6억, 비수도권 3억)을 초과하면 해당 연도부터 감면이 배제됩니다.</p>

<hr>

<h3>주의사항 및 실수하기 쉬운 점</h3>

<ul>
<li>재산세 감면은 매년 과세기준일(6.1.) 기준으로 요건 충족 여부를 판단하므로, 공시가격 변동에 주의</li>
<li>임대사업자 등록만으로 감면이 자동 적용되지 않으므로, 별도 감면 신청 필요</li>
<li>임대료 증액 제한(5%) 위반 시 재산세 감면이 소급 취소될 수 있음</li>
<li>재산세 감면은 의무임대기간 동안만 적용되며, 등록 말소 시 감면 종료</li>
<li>최소납부세액 규정이 적용될 수 있으므로 실제 감면액 확인 필요</li>
</ul>

<h3>절세 전략</h3>

<ul>
<li>전용 40m2 이하 소형 임대주택은 재산세 100% 면제 → 소형 다호수 투자 시 유리</li>
<li>취득세 감면(최초 분양 한정)과 재산세 감면(매입 포함)은 별개이므로 각각 요건 확인</li>
<li>공시가격이 기준에 근접한 주택은 공시가격 상승으로 감면이 배제될 수 있으므로 여유 있는 가격대 선택</li>
<li>재산세 감면 + 종합부동산세 합산배제 + 소득세 감면(임대소득 분리과세) 등 세제 혜택 종합 활용</li>
<li>의무임대기간 10년 동안의 총 세제 혜택을 사전에 시뮬레이션하여 투자 의사결정</li>
</ul>
