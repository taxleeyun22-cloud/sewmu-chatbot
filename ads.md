## 0. 우리 상품이 뭐냐 — 핵심 정리

### 🤖 AI 세무 상담 챗봇
- **누가 만들었나**: 대구 달서구 세무회계 이윤, 이재윤 대표세무사가 직접 운영·검증
- **어떻게 운영되나**: GPT-4.1-mini + 국가법령정보센터 API + 71개 검증된 FAQ
- **차별점**: 답변마다 **신뢰도 [높음/보통/낮음] 자동 표시** + 법조문 근거 포함. 모르는 건 "확인이 필요합니다" 솔직 답변. 할루시네이션 차단.

### 📊 사용 권한 (정확)
- **비회원**: 사용 불가 (로그인 필수)
- **카카오·네이버 1초 가입**: 일 **5회**
- **기장거래처**: **무제한**
- **거절**: 0회

### 💬 기장거래처 전용 — 카톡 같은 상담방
- 사장님·직원과 실시간 채팅 (카카오톡 톤·UX)
- 영수증·계약서·세금계산서·신분증 사진 한 번에 업로드 (10MB)
- 1년 후 사라지는 카톡과 다름 — **영구 보관 + 검색 + 카테고리·태그·D-day 메모**
- 음성 통화 · 게시판 · 첨부 파일 · 사진 갤러리

### 💰 환급 조회 (외부 연동)
- 챗봇 안에 환급 조회 버튼 — taxfriends 자동 연동
- 내가 더 낸 세금 무료 확인

### 📚 세무 칼럼 29편 (콘텐츠 자산)
- 가족간 차용·증여세, 양도소득세, 법인세, 약국·한의원·배달업·임대사업자 등 실무 분야별
- SEO 컨텐츠 + 블로그·카드뉴스·인스타 캡션 소스로 활용 가능

### 🛡️ 보안·신뢰
- 카카오/네이버 OAuth (비밀번호 X)
- 본명 확인 후 승인 (가명 차단)
- 답변 신뢰도 자동 분류 + 의심 답변 자동 검토 큐

```

매년 이맘때 가장 많이 받는 질문이 두 가지입니다.

"제가 셀프로 해도 되나요?"
"세무사한테 맡기면 얼마나 차이가 나나요?"

오늘은 솔직하게 정리해 드립니다.

━━━━━━━━━━━━━━━━━━━━

1. 신고 대상

소득세법 제70조에 따라, 다음에 해당하면 5월 안에 신고해야 합니다.

- 사업자 (개인사업자 전부)
- 프리랜서 (3.3% 원천징수 받은 분)
- 임대 소득 있는 분
- 근로소득 + 사업·기타소득 동시 발생한 분

━━━━━━━━━━━━━━━━━━━━

2. 신고 기한 + 가산세

기한: 2026년 5월 1일 ~ 5월 31일.

지키지 않으면:
- 무신고 가산세: 산출세액의 20% (국세기본법 제47조의2)
- 납부지연 가산세: 일 0.022% (연 약 8%)

━━━━━━━━━━━━━━━━━━━━

3. 셀프 신고 (홈택스) 실제 사례

저희 거래처에 오시기 전 셀프로 하셨던 사장님 한 분의 사례입니다.

- 환급 800만원 누락 (의료비·기부금 입력 빠짐)
- 가산세 47만원 (납부 늦어서)
- 5월 한 달 동안 본인 시간 30시간 이상 소요

세무사 비용보다 훨씬 큰 손실입니다.
"내가 다 알고 있다" 라고 자신하시는 분도 한 번은 비교해 보세요.

━━━━━━━━━━━━━━━━━━━━
```
